For Developers

Get the code

All SpaceScout projects are hosted on GitHub.

  • Scout server is required to get started.
  • Scout web app is the web client.
  • Scout Manager for managing food/study/tech in Scout.
  • Scout LabStats integrates with LabStats software installed in your computer labs to show students real-time computer availability in Scout.

Get it running

The README in each repository should provide step-by-step directions for how to set up a developer build. Some repositories will have docker-compose files for running the app in a Docker container.

Submitting contributions

For now, if you have contributions, just submit a pull request via GitHub. Please include unit tests if you can.

Get in touch

If you're interested in contributing or need a little help with a Scout implementation for your own institution, let us know at help@uw.edu.

If you're interested in licensing the Scout Trademark, let us know at help@uw.edu.
